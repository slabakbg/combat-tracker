# D&D Combat HP Tracker

A lightweight, single-file, offline-first combat tracker for Dungeons & Dragons & other TTRPGs.

![alt text]([https://github.com/adam-p/markdown-here/raw/master/src/common/images/icon48.png](https://github.com/slabakbg/combat-tracker/blob/main/CombatTrackerPic.png) "Screengrab")

## Features
- HP / Temp HP tracking
- Damage + healing
- Initiative sorting (manual)
- Combat log per character
- Import/export JSON
- Fully offline

## Usage
Open index.html in browser. (You can rename it, it will work)
