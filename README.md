# D&D Combat HP Tracker

A lightweight, single-file, offline-first combat tracker for Dungeons & Dragons & other TTRPGs.

![alt text](https://github.com/slabakbg/combat-tracker/blob/8716e1484c5109b1ed6c1cf03c90e827017e6d30/CombatTrackerPic.png "Screengrab")

## Features
- HP / Temp HP tracking
- Damage + healing
- Initiative sorting (manual)
- Combat log per character
- Import/export JSON
- Fully offline

## Usage
Open index.html in browser. (You can rename it, it will work)
